# Business Playbook — PH WiFi 廣告系統業務手冊

> 這份手冊幫助你從「設備安裝完畢」到「穩定獲利」。

---

## 1. 如何選擇好場地

### 1.1 人流量估算標準

目標：每個熱點每天 **500+ 連線次數**（達到經濟效益）

| 場地類型 | 每日預估人流 | 連線轉化率 | 每日連線數 | 推薦度 |
|----------|------------|-----------|-----------|--------|
| 餐廳/咖啡廳（中型）| 200-500人 | 60-80% | 120-400 | ⭐⭐⭐⭐⭐ |
| 商店街/市場 | 500-2000人 | 20-40% | 100-800 | ⭐⭐⭐⭐ |
| 診所/候診室 | 50-200人 | 80-95% | 40-190 | ⭐⭐⭐ |
| 理髮廳/美甲 | 30-80人 | 70-90% | 21-72 | ⭐⭐ |
| 車站/公車站 | 1000-5000人 | 5-15% | 50-750 | ⭐⭐⭐ |

### 1.2 現場評估 Checklist

去看場地時帶著手機，現場做這些：

- [ ] **數人流**：在尖峰時段（午飯 12-1pm、晚上 6-8pm）數 10 分鐘有多少人進出，乘以 6 = 每小時人流
- [ ] **看停留時間**：人會在這裡待多久？待越久越好（等待類場景最佳）
- [ ] **確認網路可用性**：有沒有既有 WiFi？使用者有沒有在用手機上網？
- [ ] **確認電源**：EAP 需要接電，確認場地可以提供插座
- [ ] **確認競爭**：有沒有其他免費 WiFi？（有也沒關係，我們的 WiFi 體驗更好）
- [ ] **確認 ISP**：場地有沒有固定寬帶？（需要至少 30Mbps 才夠用）

### 1.3 最佳場景排行

1. **首選：等待型場景**
   - 診所候診室、政府辦公室排隊、銀行等待
   - 人們無聊，會主動連 WiFi，且有時間看廣告

2. **次選：餐飲場景**
   - 午餐餐廳、咖啡廳、奶茶店
   - 停留 30-90 分鐘，反覆連線

3. **第三選：交通場景**
   - Jeepney 終點站、市場入口
   - 人流大但停留短，需要更多 AP 覆蓋

## 2. 如何跟場地業主談

### 2.1 分潤 vs 固定費分析

| 方案 | 適合情況 | 優點 | 缺點 |
|------|----------|------|------|
| **分潤（50/50）** | 大流量場地 | 業主有積極性維護，低初始成本 | 收入不穩定 |
| **固定費** | 小流量場地 | 收入可預測 | 業主無動力推廣 |
| **免費提供 WiFi** | 初期試點 | 最容易說服業主 | 無分潤，全靠廣告 |

### 2.2 推薦策略：先免費，後分潤

**Phase 1（前 1 個月）：** 免費提供 WiFi
> 「我幫你裝免費 WiFi，設備和網路費我出，幫你吸引客人。你什麼都不用做。」

**Phase 2（第 2 個月起）：** 提出分潤
> 「WiFi 運作得很好，每個月有 xxx 人使用。我現在有廣告收入，想跟你分潤 30%，大約每月 PHP xxx。」

### 2.3 談判話術

**開場（10 秒版本）：**
> *"Hi po, may proposal po ako para sa inyo. I can install free WiFi sa lugar niyo, para mas maraming customers na mananatili. Libre lahat ng equipment at internet. Pwede ba tayong mag-usap?"*

**中文版（台商或華人業主）：**
> 「你好，我可以免費幫你安裝 WiFi，設備和費用我來出，幫你留住客人待更久。完全免費，你覺得怎麼樣？」

**常見問題回答：**

| 業主問題 | 回答 |
|----------|------|
| 設備要多少錢？ | 我來出，你不用付任何費用 |
| 我已經有 WiFi 了 | 我幫你升級，速度更快，還可以管理連線人數 |
| 廣告是什麼廣告？ | 一般商業廣告，不影響顧客體驗，看 15 秒就可以上網 |
| 你怎麼賺錢？ | 廣告費用，每個月我也會分你 20-30% |

### 2.4 合約要點

正式合作應簽簡單合約，包含：
- 設備所有權（你的）
- 場地使用期限（建議 1 年）
- 電費誰付（場地業主付，或你補貼）
- 分潤比例和付款日期
- 雙方終止條件

## 3. 如何拜訪廣告主（直接廣告）

廣告主廣告比 Adcash 收入高 3-5 倍，是高級目標。

### 3.1 目標廣告主類型

| 類型 | 適合場景 | 預算等級 |
|------|----------|----------|
| 本地餐廳連鎖 | 附近的 Jollibee、Max's Restaurant | $$$ |
| 電信公司 | Globe、Smart（促銷方案）| $$$$ |
| 銀行/金融 | GCash、Maya、BDO | $$$$ |
| 藥妝店 | Mercury Drug、Watsons | $$$ |
| 本地市場商家 | 附近商家做在地廣告 | $$ |

### 3.2 提案格式

**一頁式提案（A4，英文+Tagalog）：**

```
PH WiFi Portal — Advertising Proposal

📍 Location: [場地名稱], [城市]
👥 Monthly Reach: [預估人數] unique visitors
📱 Ad Format: Full-screen display (15 seconds)
🎯 Audience: Local residents, age 18-45

Pricing:
- Standard (Banner): PHP [X] / month
- Featured (Interstitial): PHP [X] / month  
- Exclusive (All slots): PHP [X] / month

Contact: [你的名字] | [電話] | [Email]
```

## 4. 直接廣告主定價參考

### 4.1 定價邏輯
- 菲律賓本地廣告市場：$1-5 CPM（每千次曝光）
- 直接廣告主通常接受 2-3 倍 Adcash 價格（因為更精準、本地化）

### 4.2 建議定價

**單一熱點（1,000 人/天）：**
| 方案 | 說明 | 月費（PHP）| 月費（USD）|
|------|------|-----------|-----------|
| Bronze | 20% 廣告時段 | 2,000 | ~35 |
| Silver | 50% 廣告時段 | 4,500 | ~80 |
| Gold（獨佔）| 100% 廣告時段 | 8,000 | ~140 |

**多熱點（5 個，5,000 人/天）：**
| 方案 | 月費（PHP）| 月費（USD）|
|------|-----------|-----------|
| 網絡包（Bronze×5）| 8,500 | ~150 |
| 網絡包（Gold×5）| 35,000 | ~620 |

### 4.3 如何開發票
- 初期：使用 GCash 或銀行轉帳，附上收據
- 正式化後：辦理 BIR 登記，開具 Official Receipt

## 5. 如何擴展到多個熱點

### 5.1 擴展時機
達到以下條件時考慮擴展：
- [ ] 第一個熱點每月收入超過 PHP 2,000
- [ ] 系統穩定運作 1 個月以上
- [ ] 有第二個潛在場地

### 5.2 擴展成本
每增加一個熱點的增量成本：

| 項目 | 費用 |
|------|------|
| EAP225 或 EAP650 | PHP 2,500-4,000 |
| 場地網路費（分擔）| PHP 500-1,000/月 |
| 安裝工時 | 2-3 小時（你自己做）|
| 軟體成本 | PHP 0（已包含在系統內）|

### 5.3 多熱點管理架構

```
你的電腦 / Zeabur Cloud
       ↓
  管理員 Dashboard
  /admin/locations
       ↓
  熱點 A    熱點 B    熱點 C
  (餐廳)   (診所)   (市場)
```

每個熱點的數據都在同一個 Dashboard 可以查看。

## 6. 每日/每週運營 Checklist

### 每日（5 分鐘）

```
早上：
□ 打開 Dashboard → 確認所有熱點 Status = Online
□ 看昨日 Sessions 數量（有沒有異常下降？）
□ 確認廣告收入有在增加

晚上：
□ 看 Adcash Dashboard 今日 Impressions
□ 有沒有設備 offline 警告？（Email 通知）
```

### 每週（15 分鐘）

```
□ 週報：整理各熱點連線人數 + 收入
□ 確認 Adcash 收入是否符合預期
□ 有沒有需要拜訪的廣告主？
□ 有沒有客訴需要處理？（場地業主反映）
□ 更新場地業主（傳一個簡單數據截圖給他們）
```

### 每月（1 小時）

```
□ 計算總收入：Adcash + 直接廣告主
□ 支付場地業主分潤（如有）
□ 提領 Adcash 收入（達 $25 門檻）
□ 評估新場地候選清單
□ 是否有設備需要維護/更換？
□ 計劃下個月擴展目標
```

## 附錄：菲律賓市場特別注意事項

1. **語言**：Tagalog + English 都要準備（提案、廣告都是）
2. **支付**：GCash 最普及，場地業主分潤建議用 GCash 轉帳
3. **季節性**：聖誕假期（11月-1月）人流量最高，廣告 CPM 也最高
4. **停電**：準備 UPS 不斷電系統給 Router 和 AP（菲律賓停電頻繁），Omada Controller 在雲端 VPS 不受影響
5. **競爭**：Globe/Smart 的 Free WiFi 也在搶地盤，主打「更快更穩定」差異化
