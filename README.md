# AbotKamay WiFi

**Free public WiFi platform for the Philippines.**

Users connect to WiFi, watch a short ad, and get 10 minutes of free internet. Zero cost to users, ad revenue for operators.

## Architecture

```
                         ┌─────────────────────────┐
                         │   OC200 (Hardware Ctrl)  │
                         │   192.168.0.239          │
                         └────────────┬────────────┘
                                      │ Omada SDK API
         ┌──────────────┐             │
         │  EAP225      │─────────────┤
         │  (Indoor AP) │             │
         └──────────────┘             │
                         ┌────────────▼────────────┐
         ┌──────────────┐│                          │
         │ EAP650-Out   ││   [User connects WiFi]   │
         │ (Outdoor AP) │└────────────┬────────────┘
         └──────────────┘             │ HTTP redirect
                                      │
                         ┌────────────▼────────────┐
                         │   Portal Frontend        │
                         │   (Adcash Ad)             │
                         └────────────┬────────────┘
                                      │ POST /api/grant-access
                         ┌────────────▼────────────┐
                         │   FastAPI Backend         │
                         │   + PostgreSQL + Redis   │
                         └────────────┬────────────┘
                                      │ Authorize client
                         ┌────────────▼────────────┐
                         │   OC200 Hotspot API      │
                         │   (grant internet access)│
                         └─────────────────────────┘
```

**Traffic Flow:**
1. User connects to the open WiFi SSID (no password)
2. OC200 redirects HTTP traffic to `https://your-domain/portal`
3. Portal shows a short Adcash ad with countdown
4. After countdown, user clicks "Get Free WiFi"
5. FastAPI calls OC200 API to authorize the user's MAC address
6. User gets 10 minutes of free internet

## Quick Start

### Prerequisites

- Ubuntu 22.04 LTS VPS (1 vCPU / 1GB RAM minimum)
- Domain name pointed to your server IP
- TP-Link OC200 + EAP access points on the same network
- Adcash publisher account (see [docs/ADCASH-SETUP.md](docs/ADCASH-SETUP.md))

### Installation

```bash
git clone https://github.com/XingCEO/ph-wifi-portal.git
cd ph-wifi-system

# Run the one-click setup (Ubuntu 22.04)
sudo bash deploy/setup.sh
```

The setup script will:
- Install Docker & Docker Compose v2
- Prompt for all required configuration
- Generate `server/.env` with secure random keys
- Obtain Let's Encrypt SSL certificate automatically
- Start all services (Nginx, Next.js, FastAPI, PostgreSQL, Redis)
- Run database migrations

### Local Development

```bash
# Backend (terminal 1)
cd server
cp .env.example .env
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# Brand website (terminal 2)
cd web
npm install
npm run dev -- -p 3001
```

- Brand website: http://localhost:3001
- Portal: http://localhost:8000/portal
- Admin: http://localhost:8000/admin/

### Configuration

All configuration is in `server/.env` (auto-generated by setup.sh):

```env
# Application
DOMAIN=portal.example.com
SECRET_KEY=<auto-generated 64-char hex>
ADMIN_PASSWORD=<set a strong password>

# Omada Controller
OMADA_HOST=192.168.0.239
OMADA_PORT=8043
OMADA_CONTROLLER_ID=<from OC200 URL>
OMADA_OPERATOR=ph_wifi_operator
OMADA_PASSWORD=<operator password>

# Adcash
ADCASH_ZONE_KEY=<your zone key>

# Business Rules
AD_DURATION_SECONDS=30
SESSION_DURATION_SECONDS=600
ANTI_SPAM_WINDOW_SECONDS=3600

# Database (auto-generated)
POSTGRES_USER=ph_wifi_user
POSTGRES_PASSWORD=<auto-generated>
```

## Hardware

| Device | Model | Purpose | Notes |
|--------|-------|---------|-------|
| Controller | TP-Link OC200 | Manages all APs | Hardware controller, no cloud required |
| Indoor AP | TP-Link EAP225 | Indoor coverage | AC1200 Wi-Fi 5, ~30m radius, PoE |
| Outdoor AP | TP-Link EAP650-Outdoor | Outdoor coverage | AX3000 Wi-Fi 6, IP67 waterproof, ~50m |

**Recommended deployment:**
- 1x OC200 (connect to router LAN port)
- 2-4x EAP225 (cafe interior, lobby, waiting areas)
- 1-2x EAP650-Outdoor (parking lot, entrance, outdoor terrace)

All EAPs connect to OC200 via PoE switch.

## Deployment

### Option A: Self-hosted VPS (Recommended)

Any Ubuntu 22.04 VPS works. Recommended providers for Philippines latency:

| Provider | Region | Price | Latency to PH |
|----------|--------|-------|---------------|
| Vultr | Singapore | $6/mo | ~30ms |
| DigitalOcean | Singapore | $6/mo | ~35ms |
| Linode (Akamai) | Singapore | $5/mo | ~35ms |

```bash
sudo bash deploy/setup.sh   # Follow interactive prompts
```

Docker Compose runs 5 services: Nginx, Next.js (brand site), FastAPI (portal + admin), PostgreSQL, Redis.

### Option B: Zeabur

Zeabur handles Docker orchestration, SSL, and domain routing automatically. Note: only the FastAPI backend is deployed on Zeabur; the Next.js brand site needs separate hosting (e.g., Vercel).

### Updates (Zero Downtime)

```bash
sudo bash deploy/update.sh              # Pull latest + rolling restart
sudo bash deploy/update.sh --skip-pull  # Rebuild only, no git pull
```

### Backup

```bash
bash deploy/backup.sh                         # Local backup (keeps 7)
bash deploy/backup.sh --s3=s3://bucket/path   # + upload to S3/R2
```

# 繁體中文

## AbotKamay WiFi — 菲律賓免費公共 WiFi 平台

讓免費 WiFi 成為持續被動收入。用戶觀看一段短廣告，即可獲得 10 分鐘免費上網。用戶零費用，你賺廣告收益。

## 架構說明

```
OC200 硬體控制器 (192.168.0.239)
    ├── EAP225 x 2-4（室內覆蓋）
    └── EAP650-Outdoor x 1-2（戶外覆蓋，IP67 防水）

連線流程：
[用戶連接 WiFi]
    → OC200 重導向 HTTP 到 Portal 頁面
    → 顯示 Adcash 廣告（含倒數計時）
    → 用戶點擊「免費上網」
    → FastAPI 呼叫 OC200 API 授權 MAC 地址
    → 用戶獲得 10 分鐘免費上網
```
